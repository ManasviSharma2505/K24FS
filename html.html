HTML Tags and Elements

Some basic HTML tags include <h1>, <p>, <a>, <img>, <div>, and <span>.

  • <h1>-<h6>: Header tags define headings.
  • <p>: Paragraph tag for text blocks.
  • <a>: Anchor tag for links.
  • <b>:It is used to bold the text
  • <i>:It is used to italisize the text
  • <b>:bold tag is used to bold the text
  • <u>:Underline tag is used to undeline the text
  • <br>:is used to break line
  • <hr>:is used to give horizontal line
  • <article>:is used for writting articles.
  • <del>:is used to put linethrough.
  • <blockquote>:is used to write something in block.
  • <link>:is used to link our html file with other files
  • <audio>is used to put audio
