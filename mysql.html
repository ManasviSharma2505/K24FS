Introduction to DBMS

A Database Management System (DBMS) is software that allows users to define, create, manage, and control access to databases. It serves as an interface between the end-users and the database, enabling efficient data handling, storage, retrieval, and management. DBMSs are crucial in applications that require structured data storage, organization, and secure access.

Functions of DBMS

  • Data Definition: Creating and modifying the structure of the database using Data Definition Language (DDL).
  • Data Manipulation: Querying and updating data using Data Manipulation Language (DML).
  • Data Security: Ensuring data protection through user access controls and permissions.
  • Data Integrity: Enforcing rules to ensure data accuracy and consistency.
  • Concurrency Control: Managing simultaneous access by multiple users to prevent conflicts.
  • Backups and Recovery: Protecting data against loss by creating backups and restoring data in case of failures.

Advantages of DBMS

  • Improved data sharing and data security
  • Centralized data management
  • Reduced data redundancy and inconsistency
  • Improved data integrity
  • Efficient data backup and recovery

Disadvatages of DBMS

  • Complexity and size of the software
  • Higher cost (for some systems)
  • Potential for increased vulnerability if not managed securely

DDL Commands

  • Creating databases

    • SYNTAX:
    create database School;
    #this will create a database with name School

  • Opening databases

    • SYNTAX:
    use School;
    #this will open a database with name school

  • Removing databases

    • SYNTAX:
    drop database School;
    #this will delete a database with name school

  • Creating table

    • SYNTAX:
    create table student(rollno integer primary key,name char(30),marks integer not null,gender char(1),dateofbirth date);
    #this will create a table with name student which has rollno which is primary key that means its values can not be repeated in any other row and it should not be null.It also contains name of student which is in character datatype and has size of 30 bits.It contains marks of the student which is in integer data type.it has gender with character datatype and dateofbirth with date datatype.

  • Viewing table structure

    • SYNTAX:
    describe School;
    #this will show u the structure of te table

  • Alter table

    1. adding a column to table

      2. SYNTAX:
      alter table School add(city char(30));
      #this will add add column to the table

    2. adding a column to table with default value

      3. SYNTAX:
      alter table School add(city char(30) default "Jalandhar");
      #this will create column with defaut value as Jalandhar

    3. modifying existing column definition

      4. SYNTAX:
      alter table School modify(name char(50));
      #this will change size of character datatype of name in a table

    4. rename a column

      5. SYNTAX:
      alter table School change(city state char(30));
      #this will change name of column from city to state

    5. removing a column from table

      6. SYNTAX:
      alter table School drop(state);
      #this will remove state column from the table

  • drop table command

    • SYNTAX:
    drop table student;
    #this will delete the table student

DML commands

  1. inserting data into table

    2. SYNTAX:
    insert into student values(1,"Rahul","M","88","2002-11-12"); insert into student values(2,"Vedika","F","89","2003-08-10"); insert into student values(3,"Yashika","F","88","2002-05-26"); insert into student values(4,"Ankit","M","82","2002-12-19");

  2. modifying data in a table

    3. SYNTAX:
    update student set marks=90 where name="Rahul";
    #This will update the marks of student where name is rahul

  3. updating multiple columns

    4. SYNTAX:
    update student set marks=90 and rollno=2 where name="Rahul";
    #This will change the marks and rollno of student where name is rahul

  4. Updting to null value

    5. SYNTAX:
    update student set marks=NULL where name="Rahul";
    #This will set marks of student as null where name is rahul

  5. removing data in a table

    6. SYNTAX:
    delete from student where name=rahul;
    #This will delete row from table

DQL(Data Query Language) COMMANDS

  1. select command

    2. SYNTAX:
    select name,gender from student;
    #this will show you a table containing name and gender of students

  2. selecting complete table

    3. SYNTAX:
    select * from student;
    #This will show you complete table

  3. Eliminating redundant values

    4. SYNTAX:
    select distinct marks from student;
    #this will show u a table containing distinct marks only that is it will not show you marks 88 two times.

SQL Operators
